# Querying the MFEM Laghos dataset with DuckDB
Querying with DuckDB is trivial

## Install DuckDB
DuckDB is available at the following address:
```
wget https://github.com/duckdb/duckdb/releases/download/v0.9.2/duckdb_cli-linux-amd64.zip
unzip duckdb_cli-linux-amd64.zip
mv duckdb ~/bin/duckdb
```

## Prepare the dataset
gunzip taproot/data/1m_points/1m_points.parquet.gz

# Analyze data with DuckDB
duckdb
select * from 'taproot/data/1m_points.parquet' limit 10;
